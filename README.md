# 📓 Dev Diary

Dev Diary is a personal blog and learning platform where I document my journey as a web developer.  
It includes blogs, a dashboard system, and multiple pages like About, Contact, Support, Profile, and Revenue.

---

## 🚀 Live Features

- 🏠 Home page with modern landing design
- 📝 Blog system with dynamic data
- 📊 Dashboard with sidebar navigation
- 👤 Profile page with user details
- 💰 Revenue tracking page
- 📩 Contact form UI
- 🛠️ Support page
- 📖 About page

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React.js
- Tailwind CSS
- JavaScript (JSX)
- Responsive UI design

---

## 🎯 Purpose

The main goal of this project is to:

- Improve my frontend development skills
- Learn Next.js routing and layout system
- Practice UI/UX design with Tailwind CSS
- Build a real-world portfolio project

---

## 📌 Pages

- `/` → Home

- `/contact` → Contact form
- `/support` → Help & support
- `/dashboard` → Admin dashboard

- `/dashboard/revenue` → Revenue analytics
